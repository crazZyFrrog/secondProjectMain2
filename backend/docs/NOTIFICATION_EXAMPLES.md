# 📱 Примеры Telegram-уведомлений

## Как выглядят уведомления в Telegram

### 1. Уведомление о регистрации нового пользователя

```
🎉 Новая регистрация пользователя

👤 Имя: Иван Иванов
📧 Email: ivan@example.com
🏢 Тип компании: Малый бизнес
🆔 ID клиента: a1b2c3d4-e5f6-7890-abcd-ef1234567890
📦 Текущий тариф: Free
```

**Когда отправляется:**
- При регистрации нового пользователя через фронтенд
- При регистрации через API `/api/auth/register`

---

### 2. Уведомление о переходе на Pro тариф

```
💎 Новый клиент оформил тариф Pro

📦 Тариф: Pro
📧 Email: user@example.com
🆔 ID клиента: a1b2c3d4-e5f6-7890-abcd-ef1234567890
```

**Когда отправляется:**
- При переходе с Free на Pro тариф
- При переходе с Starter на Pro тариф
- При выборе Pro тарифа через API `/api/clients/me/plan`

---

### 3. Уведомление о переходе на Enterprise тариф

```
👑 Новый клиент оформил тариф Enterprise

📦 Тариф: Enterprise
📧 Email: enterprise@company.com
🆔 ID клиента: a1b2c3d4-e5f6-7890-abcd-ef1234567890
```

**Когда отправляется:**
- При переходе на Enterprise тариф
- При выборе Enterprise тарифа через API `/api/clients/me/plan`

---

### 4. Тестовое сообщение

```
Тестовое сообщение от Landing Constructor API
```

**Когда отправляется:**
- При вызове API эндпоинта `/api/integrations/telegram/test-message` (только админ)
- При запуске скрипта `python test_telegram.py`

## Особенности форматирования

### Используемые элементы

- **Emoji:** 🎉 💎 👑 👤 📧 🏢 🆔 📦
- **Жирный текст:** `<b>текст</b>` → **текст**
- **Моноширинный шрифт:** `<code>текст</code>` → `текст`

### HTML-теги поддерживаемые Telegram

В сообщениях используется `parse_mode: "HTML"`, что позволяет:

- `<b>жирный</b>` или `<strong>жирный</strong>`
- `<i>курсив</i>` или `<em>курсив</em>`
- `<code>моноширинный</code>`
- `<pre>код блок</pre>`
- `<a href="url">ссылка</a>`
- `<u>подчёркнутый</u>`
- `<s>зачёркнутый</s>`

## Настройка внешнего вида

### Изменение текста сообщений

Откройте файл `backend/telegram_notifications.py`:

```python
# Найдите функцию send_new_user_registration_notification
def send_new_user_registration_notification(...):
    text = (
        "🎉 <b>Новая регистрация пользователя</b>\n\n"
        f"👤 Имя: <b>{username}</b>\n"
        # ... измените текст здесь
    )
```

### Добавление новых emoji

Полный список emoji: https://emojipedia.org/

Примеры полезных emoji:
- 💰 🎯 ⚡ 🚀 ✅ ❌ ⚠️ ℹ️
- 📊 📈 📉 💻 🖥️ 📱 💳 🔔
- 👥 👤 🏢 🏪 🏭 🏗️ 🌐 📍

### Изменение форматирования

```python
# Примеры форматирования:
text = "<b>Жирный текст</b>"
text = "<i>Курсив</i>"
text = "<code>Моноширинный</code>"
text = "<a href='https://example.com'>Ссылка</a>"
text = "<b>Жирный</b> и <i>курсив</i> вместе"
```

## Группировка уведомлений

Telegram автоматически группирует сообщения от одного бота, отправленные в течение короткого времени.

Пример:
```
┌─────────────────────────────┐
│ Landing Constructor Bot     │
├─────────────────────────────┤
│ 🎉 Новая регистрация...     │
│ 14:35                       │
├─────────────────────────────┤
│ 💎 Новый клиент оформил...  │
│ 14:36                       │
├─────────────────────────────┤
│ 🎉 Новая регистрация...     │
│ 14:37                       │
└─────────────────────────────┘
```

## Отключение уведомлений

### Для бота (все уведомления)

В Telegram:
1. Откройте чат с ботом
2. Нажмите на имя бота вверху
3. Выберите "Отключить уведомления"

### Для определённых типов (будущая функция)

В следующих версиях планируется добавить настройку типов уведомлений через UI приложения.

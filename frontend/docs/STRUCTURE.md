## 📊 Управление состоянием

### Auth Store (`src/store/authStore.ts`)
**Zustand store для авторизации**

**Состояние:**
```typescript
{
  user: User | null,
  isAuthenticated: boolean
}
```

**Методы:**
- `login(email, password)` - вход
- `signup(name, email, password, plan)` - регистрация
- `logout()` - выход

**Хранение:** localStorage (`user`, `token`)

---

### Project Store (`src/store/projectStore.ts`)
**Zustand store для проектов**

**Состояние:**
```typescript
{
  projects: Project[],
  currentProject: Project | null
}
```

**Методы:**
- `addProject(project)` - создать проект
- `updateProject(id, updates)` - обновить проект
- `deleteProject(id)` - удалить проект
- `setCurrentProject(id)` - установить текущий
- `getProjectById(id)` - получить по ID

**Хранение:** localStorage (`projects`)

- **Primary:** оттенки синего (#0ea5e9 - #0c4a6e)

- **Warning:** желтый

### Компоненты
- Кнопки: primary (синие), secondary (серые), danger (красные)
- Карточки: белый фон, тень, скругленные углы
- Формы: border-gray-300, focus:ring-primary-500
- Модальные окна: backdrop blur, центрирование

### Иконки
- Библиотека: Lucide React
- Размеры: 16-24px для UI, 32-48px для feature секций

## 🚀 Основные флоу пользователя

### Флоу 1: Новый пользователь
1. Заходит на `/` → видит главную
2. Клик "Начать бесплатно" → `/signup`
3. Регистрация → автоматический вход → `/dashboard`
4. Видит empty state "У вас пока нет проектов"
5. Клик "Создать проект" → `/projects/new`
6. Выбирает шаблон + название → создает
7. Попадает в редактор `/projects/:id/edit`
8. Заполняет форму "О компании"
9. Клик "Превью" → `/projects/:id/preview`
10. Клик "Экспорт" → `/projects/:id/export`

### Флоу 2: Использование AI
1. В редакторе клик "AI-генерация" → `/projects/:id/ai-generate`
2. Заполняет форму о компании
3. Клик "Сгенерировать контент" → ждет 3 сек
4. Видит сгенерированный контент
5. Клик "Применить к проекту" → возврат в редактор
6. Данные автоматически заполнены

### Флоу 3: Работа с шаблонами
1. Клик "Шаблоны" в меню → `/templates`
2. Фильтрует по категории (например, "IT")
3. Клик на карточку → модальное окно с полным превью
4. Клик "Использовать шаблон"
5. Если не авторизован → `/login` → после входа → `/projects/new`
6. Если авторизован → сразу `/projects/new` с предвыбранным шаблоном

## 💾 Структура данных проекта

```typescript
Project {
  id: string
  name: string
  templateId: string
  createdAt: string
  updatedAt: string
  status: 'draft' | 'completed'
  thumbnailUrl: string
  data: {
    company: {
      name: string

      description: string
      mission: string

    }>

      id, icon, title, description

    }>

    }
    cases: Array<{
      id, title, client, challenge, solution, results, images[]
    }>
    faq: Array<{
      id, question, answer
    }>
  }
}
```

# Changelog V2

## Изменения от V1 к V2

### Измененные файлы:

#### 1. `src/pages/HomePage.tsx`
**Что изменилось:**
- **Убран блок CTA** "Готовы создать свой первый лендинг?" - удалена секция с повторным призывом к действию
- **Добавлен демонстрационный шаблон** - новая секция с полноценным примером лендинга TechSolutions (hero, услуги, футер)

**Зачем:** Показать пользователям реальный пример готового лендинга вместо повторного призыва к регистрации

---

#### 2. `src/components/Header.tsx`
**Что изменилось:**
- **Убрана иконка логотипа** - удален синий квадрат с буквой "L", оставлен только текст "LandingBuilder"
- **Упрощена навигация профиля** - убрана отдельная иконка настроек, добавлено выпадающее меню по клику на имя пользователя
- **Добавлен dropdown menu** - при клике на имя открывается меню с "Настройки" и "Выйти"

**Зачем:** Упростить интерфейс и сделать управление профилем более интуитивным через единую точку входа

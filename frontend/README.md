# Landing Page Constructor (Конструктор Лендингов)

Конструктор лендингов и коммерческих предложений с AI-генерацией контента.

## Возможности

- 🎨 База профессиональных шаблонов
- ✨ AI-генерация контента
- 📥 Экспорт в PDF, HTML, DOCX
- 🔐 Система авторизации
- 💎 Freemium модель монетизации
- 📱 Адаптивный дизайн

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   └── Header.tsx      # Шапка сайта с навигацией
├── pages/              # Страницы приложения
│   ├── HomePage.tsx           # Главная страница
│   ├── LoginPage.tsx          # Страница входа
│   ├── SignupPage.tsx         # Страница регистрации
│   ├── TemplatesPage.tsx      # Галерея шаблонов
│   ├── PricingPage.tsx        # Тарифы
│   ├── DashboardPage.tsx      # Дашборд с проектами
│   ├── CreateProjectPage.tsx  # Создание проекта
│   ├── EditorPage.tsx         # Редактор проекта
│   ├── PreviewPage.tsx        # Превью лендинга
│   ├── ExportPage.tsx         # Экспорт документов
│   ├── SettingsPage.tsx       # Настройки аккаунта
│   └── AIGeneratePage.tsx     # AI-генерация контента
├── store/              # State management (Zustand)
│   ├── authStore.ts    # Управление авторизацией
│   └── projectStore.ts # Управление проектами
├── data/               # Mock данные
│   └── templates.ts    # Шаблоны лендингов
├── App.tsx             # Главный компонент с роутингом
├── main.tsx            # Entry point
└── index.css           # Tailwind CSS

```

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **React Router v6** - роутинг
- **Tailwind CSS** - стилизация
- **Zustand** - state management
- **Lucide React** - иконки
- **Vite** - сборщик

## Установка и запуск

### 1. Установите зависимости:

```bash
npm install
```

### 2. Запустите dev-сервер:

```bash
npm run dev
```

### 3. Откройте в браузере:

```
http://localhost:5173
```

## Навигация по приложению

### Публичные страницы:
- `/` - Главная страница
- `/login` - Вход в аккаунт
- `/signup` - Регистрация
- `/templates` - Галерея шаблонов
- `/pricing` - Тарифы

### Приватные страницы (требуют авторизации):
- `/dashboard` - Дашборд с проектами
- `/projects/new` - Создание нового проекта
- `/projects/:id/edit` - Редактор проекта
- `/projects/:id/preview` - Превью лендинга
- `/projects/:id/export` - Экспорт документа
- `/projects/:id/ai-generate` - AI-генерация контента
- `/settings` - Настройки аккаунта

## Основные функции

### Авторизация
- Вход/регистрация с mock данными

- Protected routes для приватных страниц

- Создание проектов на основе шаблонов
- Редактирование информации о компании
- Автосохранение (UI only)
- Удаление проектов

### Шаблоны
- 6 готовых шаблонов
- Фильтрация по категориям
- Free и Premium шаблоны

### Экспорт
- Выбор формата (PDF, HTML, DOCX)
- Настройки экспорта
- История экспортов (mock)

### AI-генерация
- Форма для ввода данных о компании
- Mock генерация контента
- Применение к проекту

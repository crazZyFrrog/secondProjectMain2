## Запуск (3 шага):

### 1. Запустите Backend:
```bash
cd backend
.venv\Scripts\activate
python -m uvicorn main:app --reload --port 5001
```

### 2. Запустите Frontend:
```bash
cd frontend
npm run dev
```

### 3. Запустите тесты:
```bash
cd frontend
npm run test:ui
```

## Первый тест:
В UI выберите `00-проверка-настройки.spec.ts` и запустите ▶️
